# Визуализатор графа зависимостей пакетов

## Общее описание

CLI-приложение для анализа и визуализации графа зависимостей пакетов. Инструмент поддерживает работу с различными менеджерами пакетов и репозиториями.

**Текущий этап**: Этап 1 - минимальный прототип с конфигурацией

## Функциональность

### Реализовано в Этапе 1:
- ✅ Парсинг аргументов командной строки
- ✅ Валидация всех входных параметров
- ✅ Вывод конфигурации в формате ключ-значение
- ✅ Обработка ошибок с понятными сообщениями
- ✅ Подготовка структуры для следующих этапов

### Планируется в следующих этапах:
- Этап 2: Сбор данных о зависимостях
- Этап 3: Рекурсивный анализ транзитивных зависимостей
- Этап 4: Визуализация графа
- Этап 5: Интеграция и финальная доработка

## Установка и запуск

### Требования
- Python 3.7 или выше

### Запуск приложения
```bash
python main.py --package <имя_пакета> --repo <репозиторий> --mode <режим> --version <версия> --output <файл>
