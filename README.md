# Визуализатор графа зависимостей пакетов Ubuntu

## Общее описание

CLI-приложение для анализа и визуализации графа зависимостей пакетов Ubuntu. Инструмент работает с официальными репозиториями Ubuntu и использует формат APT.

**Текущий этап**: Этап 3 - основные операции над графом зависимостей

## Функциональность

### Реализовано в Этапе 1:
- ✅ Парсинг аргументов командной строки
- ✅ Валидация всех входных параметров
- ✅ Вывод конфигурации в формате ключ-значение
- ✅ Обработка ошибок с понятными сообщениями

### Реализовано в Этапе 2:
- ✅ Загрузка данных из репозиториев Ubuntu
- ✅ Парсинг файлов Packages.gz
- ✅ Извлечение информации о зависимостях
- ✅ Обработка формата APT зависимостей
- ✅ Вывод прямых зависимостей в консоль
- ✅ Сохранение данных в JSON формате

### Реализовано в Этапе 3:
- ✅ Построение графа зависимостей алгоритмом DFS с рекурсией
- ✅ Поддержка максимальной глубины анализа
- ✅ Фильтрация пакетов по подстроке
- ✅ Обнаружение и обработка циклических зависимостей
- ✅ Режим тестирования с файловым репозиторием
- ✅ Вывод ASCII-дерева зависимостей
- ✅ Сбор статистики по графу

## Установка и запуск

### Требования
- Python 3.7 или выше
- Установленные зависимости: `pip install -r requirements.txt`

### Запуск приложения
```bash
python main.py --package <имя_пакета> --repo <репозиторий> --mode <режим> --version <версия> --output <файл>
